
## Movies Lib

- Um projeto consumindo dados da API: The Movie Database (TMDB). 

## Recursos

- Um catálogo de filmes.
- Veja os detalhes de cada filmes, como: capa, avaliação, data de lançamento e um breve resumo.
- Pesquise por um filme.
- Design responsivo

## Tecnologias

- HTML
- CSS
- JavaScript
- Sass
